Robo Advisor

From SQLY Capital

We provide automatic solutions for: 1) Professional Assessment for Portfolio 2) Portfolio Recommendation Tailored for You 3) Performance of Suggested Portfolio

Existing Portfolio

The first function provides the return and risk profile of a portfolio given by the user, with specific weights and stocks. In order to complete this function, Fama-French 5 factors model is used to estimate the past returns and volatility of stocks in order to back test the portfolio rate of return and volatility.

Enhance existing portfolio n

The second function of Robo-Advisor is to enhance the existing portfolio by providing an optimal portfolio. Mean-Variance Optimization is implemented under a multi-period framework to provide the optimal portfolio which will span the userâ€™s entire investment horizon with given rebalancing frequency. The multi-period framework allows us to take the future stock returns into account so that a better portfolio can be provided. Thus, future stock returns and volatilities will be computed by ARIMA and GARCH in order to implement this model.

With Target Return

The third function is to provide an optimal portfolio with a given return goal. The same processes are used as the second function.
